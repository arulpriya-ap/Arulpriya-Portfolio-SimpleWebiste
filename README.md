# Simple Portfolio Website

This is a simple portfolio website created using **HTML and basic CSS**.  
The purpose of this project is to practice basic web development concepts and build a simple multi-page website.

## Project Overview
The website contains two main pages:
- **Home Page** – Displays basic introduction and information.
- **Contact Page** – Provides contact details or a simple contact form for communication.

## Features
- Built using HTML5 and basic CSS
- Simple and clean layout
- Easy navigation between pages
- Beginner-friendly project

## Technologies Used
- HTML5
- CSS3
